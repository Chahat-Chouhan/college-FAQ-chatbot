# college-FAQ-chatbot
GUI-based FAQ chatbot using Python and Tkinter

A GUI-based chatbot that answers college-related questions using keyword-based intent matching.

## Features
- GUI using Tkinter
- Keyword-based NLP system
- Predefined FAQ responses
- User tracking with JSON
- Quick access buttons

## Technologies Used
- Python
- Tkinter
- JSON
- Regex

## How to Run
python college_chatbot.py
